# 开源的价值在哪里?
为什么很多程序员喜欢将自己的项目放在github上供其他程序员随意使用?

这种不求回报的思想到底是怎么产生的?

虽然我也会将自己的一些项目放到github上供其他人下载使用,但是我好想没有领悟其中的精髓?

下面我能想到的几个答案

1. 使用的人数越多,可以借助其他程序员帮助完善项目
2. 使用的人数越多,这个项目越出名,作者可以获得荣誉
3. 可以获得其他的潜在收益
4. 可以在项目初期没有资源的情况下迅速获得,风投的青睐
5. 仅仅是喜欢共享有价值的东西
